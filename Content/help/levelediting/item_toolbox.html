Level editing > Item Toolbox

Item toolbox - The special tool, what contains lists of available items for place into current level section.

For switch current mode into placing item mode, you need only press on necessary item, and you can place items into level section.

For convenience, implemented groups, what will separate large item packs, categories, what separating items by his type and functions.

 

Notes:
- For display current custom graphics of current level, you can select category field of your current list into the [custom].
- If level haven't customized items, [custom] list will be empty.
- If you create new level, for apply recent updated/added graphics, you can reload current level file.
