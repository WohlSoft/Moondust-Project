NPC Properties

Each NPC have switches and flags:

  • Default direction - NPC will start his movement to defined direction. Direction option can have other type and values, for example, name "Activate" and values "On"/"Off", etc.

  • Friendly - NPC will not communicate with playable chatacter and with other NPC's. Friendly NPC can'e be killed, can't hurt to player, can't be taken or grabbed. Playable character and other NPC's can't stay on top of friendly NPCs.

  • Not movable - NPC will be idle irrespective of its algorithm.

  • Set as boss - (inside SMBX this option named as "Legacy Boss") This is a special flag, which automatically apply special events for the some bosses, what support them. For example: On activate NPC with this flag will be changed music to boss's own theme, and if boss was defeat will be spawned the special exit item.

  • Talk message - The message what will be displayed if player try to talk with this NPC. After displaying message will be activated event slot. 

  • Generator - Make the NPC generator. From one point will be spawned new NPC's every each defined time delay.

    • Generator type - Warp - NPC will be smoothly warped. Projectile - NPC will be appeared by sharply shoot.

    • Generator direction - defining the direction of spawn of NPC

Some NPC can have special values:

  • Contents - this option have a ID of other NPC, what included in this NPC. Available for NPC with marker "containers".

  • Spin box with numeric value - Some NPC can have the numeric special value, for example, position of firebar part.

  • Special combo-box - some NPC can have switched or modes of his working, what available to select from list of this modes.

 

  • Layer - here defined the layer, what is owner of this NPC. All items at first are a members of "Default" layer.

  • Attach layer - This is special option, will move all items on selected layer with movement of NPC.

Each NPC have the event slots:

Activate - This slot activating then NPC will appear on player's screen

Death - This slot activating then NPC will be defeat, destroyed, kicked or taked (coins, power-ups, etc).

Talk - This slot activating them player call to talk with this NPC.

Layer is empty - (In SMBX this option named as "No more objects in layer") this slot activating if was activated the "Death" slot and if on same layer, which is an owner if this NPC, haven't other objects.

NPC-Containers: how to use

All, who working with SMBX know "Egg", "Buried", "Bubble", "Lakuti" flags. This flags including selected item in the special NPC - which named as Container. Container can have included NPC's what can be extracted on destroy of them or will be spawning by container before it will destroyed.

 

In SMBX we pressing the "Bubble=Yes" or "Buried=yes" for place "Packed into the herb" or "Packet into the bubble" NPC.

 

In PGE-Editor you can edit NPC-Containers. And because, for example, Herb will be more useful in SMB2 group, because "The Lakutus are throwing Lakitus!" in SMBX - is a secret tab and inside SMBX need to made a lot of action for open them. Even in PGE-Editor same "tab" is non-secret "group", for place "burred" items, we must select "herb" and set its content before placing or to already placed. In PGE secrets are absence - all Free and Open Source.

Generators

Generator flag in the Properties dialog will made selected NPC as generator

 

 

NPC will be appeared every time deleay by two method: warp and projectile.

Warp - NPC will be appeared smoothy and will start it's movement on appear.

Projectile - NPC will be sharply shooted from generator.
