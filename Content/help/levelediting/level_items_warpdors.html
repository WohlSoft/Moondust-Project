Level editing > Level items > Warps and doors

Warps - is a special units, which allow to player teleport from first warp point (Entrance) to second (Exit). Player can teleport between difference places of one section, and also player can teleport between sections. This is a one way for enter to another section, but exclusion is some NPCs, what can generate warp to other section.

Warp properties

Warps have standard flags:

No Vehicles -  (In SMBX this option named as "No Yoshi") When you enter in this warp, playable character will exit without his vehicle (Vehicle will be restored when player will end this level).

Allow items - (In SMBX this option named as "Allow NPC") With this flag, playable character can get carryed item through this warp. Without this flags item will leaved nearly entrance of this warp.

Locked - With this flag warp will be locked and for enter in them, player character must have a key.

 

Special flags:

Level entrance - this option allow to place only one point. With this flag impossible to enter in this warp, and this warp can be used only for enter into this level

Level exit - this option allow to place only one point. When playable character enter into this warp, level will be done.

 

Values and options:

Stars needed - This is limit for star number. Player can enter in this warp only if he(she) collect necessary number of stars.

 

X and Y - This is a coordinated of world map. If they defined, this warp will be as exit into the world map coordinates.

 

Warp type - Defining of warp type: Instant, Pipe or Door.

Pipe directions - This is a directions of warp, which can be used only for "pipe" warp type.

 

Level file name - Here defining the another level file for make warp to them.

Door # - (Inside SMBX this option named "Warp to"). If value is 0 (Normal entrance), player will start defined level from his(her) start point, else player will entered in level by selected warp point.

Warp types

The warps have a three types: Instant, Pipe and Door.

 

Instant - Player will be instantly teleported on any contact with entrance warp point.

 

 

Door - Player can enter into this warp only if contact with entrance warp point and will press up key.

 

 

 

Pipe - The directional warp. For enter in this warp need contact with entrance warp point and press the key equal to defined direction. Player will be exited from other side wit defined exit direction.

Level entrance and exit

This is a special flags what allow to place only one warp point.

 

Level entrance - allows to place only exit point. This warp can be used only for enter in level and haven't the entrance point.

 

Level Exit - allows to place the entrance point only. The level will be end when player will enter in this warp.

Warp to another level

Player will be teleported in another level if he(she) enter in this warp.

You must simply define the level filename in the Level file field. Also, you can browse the level file and select necessary from list

The number of door will say, where player must be entered. If value is a zero - player will be start level from his(her) start point.
