Console tools

The useful tools which need for make some work with data files of your games.

 

LazyFix tool

The powerful tool will help to fix lazily-made masks of SMBX GIF pairs.

 

 

SMBX GIFs pair to PNG Converter

This tool quickly create the PNG images from SMBX GIF pairs.

 

 

 

PNG to SMBX GIFs pair

This tool is useful for graphical designers who testing own graphics in the SMBX.

This tool quickly create the GIF pair with mask. The main feature - counting alpha-channel, and target image will have almost same view which have original PNG image.
