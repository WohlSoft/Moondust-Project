Most of features are available on the toolbars and in the menus on top of a main window.

Level

  • Section settings allows You to resize sections, As well as set music, background and properties
  • Item toolbox provides a completed list of available to place elements with category/group and search filters
  • Tileset item box provides a convenient, customizable and flexible item set. If you prefer SMBX's Item Box, use this item box instead of regular item box!
  • Warps and Doors provides management of warp entries to connect different level areas.
  • Layers manages Layers which allows group item manipulation with events or scripts
  • Events manage of Evens to exchange gameplay of your levels and manipulate with layer member items
  • Search gives the ability to find any interested item on the level!

World map

  • World settings allows to configure episode settings
  • Item toolbox provides completed list of available to place elements with category/group and search filters
  • Tileset item box provides convenient, customizable and flexible item set. If you like old SMBX's item box more, use this item box instead of regular item box!
  • Search gives able to find any interested item on the world map!
Use context menu to get access to properties of target item or selected item group.

Are you don't know what is a context menu? Context menu is a menu which pop-ups when you press a right (left for lefthander) mouse button.
When you can't find your actual section or some zone on huge map, you can quickly return back to left-top or left-bottom of level section or to 0-0 position on the world map.
Position bookmarks is a good friend which remembers your screen position where you can quickly return back later.
Do you remember the bug with placing vines in SMBX? You can clean-up your level from junk NPCs caused by this bug by the menu item:
Tools ->Custom data ->Remove NPC garbage...
If you are creating graphics for SMBX, you can use PNG2GIFs tool with drag & drop interface to quickly convert your PNG images into SMBX compatible format which uses GIF images with a bit mask.
You can share your ideas and creations to public with exporting of level or world map parts into the image files.
To export part of level or world map screen you can press F12 key or open File -> Export to image menu item.
Note: To export entire level section you need to open File -> Export section to image... menu item.
Do you hear your musics playing too fast or plays buggy?
Be sure that your musics have a sample rate of 44100. If you hear that music plays too fast or buggy - your music file has a non-44100 hz sample rate. You can quickly fix your music file with the menu item: Tools ->External tools ->Convert audio (SoX)...
You can use middle mouse button for advanced convenience!
  • If no selected elements on a map, you will toggle into hand-drag mode
  • If selected one element, you will switch "placing new" mode of same element
  • If selected multiple elements, there are will be duplicated to current mouse position relative to left-top corner of selection area
You have the Overwrite mode which allows you to place new elements over existing ones with removing of them. An icon will appear on the toolbar when you will pick up an item to place.
Did you know that layers and events in a list are movable? Drag items in the list to change order of them!
Are your custom images are semi-transparent? That may be happened if your custom images are have non Black & White mask or just having a lazily-made mask.
To quickly fix them you can try:
  • Run Tools ->Custom data ->Fix wrong masks... menuitem when you have opened level or world map with broken graphics
  • Try to enable View -> Enable GFX-Fix flag in the editor and reload your level with F5 key.
In next time use PNG2GIFs tool to create SMBX specific images instead of lazily-made sprites usage.
You have a placement tools which you can find on the toolbar:
  • Rectangle allows to place bunch of items by rectangles drawing.
  • Line allows to place bunch of items by line drawing. Useful to draw columns, slopes, bridges, walls, gates, etc.
  • Flood-fill allows filling of a closed zone. Don't worry if you will fill outer space, you can undo your mistake with Ctrl+Z key combo
Note: Those buttons are will be available when you will pick up an item to place.
If you want to place elements more accurate, you can zoom in and disable grid aligning with toggling of Snap to grid switch.
Use mouse wheel and holden Alt key to zoom in or zoom out.
